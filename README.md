### 1. Вывод закономерности последовательности

Задача – посмотреть, какие значения принимает последовательность при разных a0 и n, вывести закономерность, по которой строится последовательность и запрограммировать ее самостоятельно.

### 2. Максимальное число подъёмов

Дана последовательность из n чисел. Подряд идущие элементы одинаковой четности считаем “частью”. Для каждой полученной части нужно определить, сколько раз внутри нее встречаются подъемы – текущий элемент строго больше предыдущего. Выведите наибольшее встретившееся число подъемов среди всех частей.

### 3. Вывод самой младшей нечётной цифры

С клавиатуры вводится натуральное число N. Вывести на экран самую младшую нечетную цифру в десятичном представлении числа. Если в числе нет нечетных цифр, вывести "NO".
